## 1. 화면 구조 개요

### 1.1 화면 플로우

```
┌─────────────────────────────────────────────────────────────────────┐
│                           앱 시작                                    │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        스플래시 화면                                 │
│                      (로고 + 로딩)                                   │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                    ┌─────────────┴─────────────┐
                    ▼                           ▼
            ┌─────────────┐             ┌─────────────┐
            │   로그인    │             │   메인 홈   │
            │  (미인증)   │             │  (인증됨)   │
            └─────────────┘             └─────────────┘
                    │                           │
                    ▼                           │
            ┌─────────────┐                     │
            │  온보딩     │                     │
            │ (신규유저)  │                     │
            └─────────────┘                     │
                    │                           │
                    └───────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                         메인 탭 구조                                 │
├─────────────┬─────────────┬─────────────┬─────────────┬─────────────┤
│    홈       │   앨범      │   업로드    │   알림      │   설정      │
│  (타임라인) │  (월별)     │   (카메라)  │  (활동)     │  (프로필)   │
└─────────────┴─────────────┴─────────────┴─────────────┴─────────────┘
```
