### 응답

<details>
<summary>사고 과정 (Thinking)</summary>

```
SessionEnd 훅에서 JSONL 파일을 생성하고 있습니다. 사용자는 MD 파일만 생성하고 JSONL은 생성하지 않기를 원합니다.

1. JSONL 파일로 transcript 복사
2. Ruby 스크립트로 JSONL을 MD로 변환

변경해야 할 것:
- JSONL 파일 복사를 제거하고 MD 파일만 생성하도록 수정
- transcript_path를 직접 Ruby 스크립트에 전달하여 MD 파일만 생성

하지만 Ruby 스크립트가 JSONL을 입력으로 받는 것 같으니, 임시 파일을 사용하거나 스크립트를 수정해야 할 수 있습니다. 가장 간단한 방법은:
1. 임시 파일로 transcript 복사
2. MD 변환
3. 임시 JSONL 삭제

또는 Ruby 스크립트가 직접 transcript_path를 읽을 수 있다면 그렇게 하면 됩니다.

일단 JSONL을 저장하지 않도록 하고, MD만 생성하는 방식으로 수정하겠습니다.
```

</details>



---

### 응답
